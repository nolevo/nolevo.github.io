喜获电脑一台

发表于 2019-01-03 | 阅读量 次

一个偶然的机会同一个收破烂的大娘60元购买过来的（20181230），电脑配置以下：

电脑型号 X64 兼容 台式电脑
操作系统 Windows 7 旗舰版 64位 SP1 ( DirectX 11 )
处理器 AMD A8-5600K APU with Radeon HD Graphics 四核
主板 华硕 F2A85-M LE (AMD K15)
内存 4 GB ( 威刚 DDR3 1600MHz )
主硬盘 西数 WDC WD10EZEX-00ZF5A0 ( 1 TB / 7200 转/分 )
显卡 ATI Radeon HD 7560D ( 768 MB / 华硕 )
显示器 LG GSM5825 E2281 ( 21.7 英寸 )
声卡 瑞昱 ALC887 @ ATI 高保真音频
网卡 瑞昱 RTL8168F PCI-E Gigabit Ethernet NIC / 华硕

2013的机子，不包含显示器。呵呵！购买过来时是开不了机的，给维修师傅检查发现是主板的问题。还好可以维修，换了一个电子就成功开机。维修费80块，值。一共140有台二手配置还不差的电脑用。
